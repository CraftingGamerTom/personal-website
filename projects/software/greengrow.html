Green Grow+ Mod (For Minecraft)

The Problem

I have enjoyed many Modifications made for Minecraft on the PC. I'm mostly interested in modpacks (collection of mods in use together) that mix economy and business with growing crops. I always wished that one of my favorite mods, Psychedelicraft, which introduces marijuanna and tobacco to the game, had different kinds of marijuanna. So I was going to make a mod that had new strains and colors.

Type Requirement
Minecraft Version 1.7.10
Development Stage Alpha

Solution

This was my first real java project. I learned about building with gradle and began learning how to better organize code. I wrote the application, implementing multiple new kinds of plant alternatives.

I decided to move away from the project. I does have a few bugs but I was successful in learning a lot about programming and object oriented programming.

The code can still be found on my Github.
